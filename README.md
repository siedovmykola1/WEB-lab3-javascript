# Лабораторна робота №3: Генератор паролів на JavaScript

## 📝 Опис проєкту
Цей вебзастосунок дозволяє користувачам генерувати надійні та безпечні паролі. Користувач може обрати довжину пароля (від 4 до 20 символів) та типи символів:
* Малі літери (a-z)
* Великі літери (A-Z)
* Цифри (0-9)
* Спеціальні символи (!@#$%^)

Також реалізована функція швидкого копіювання згенерованого пароля в буфер обміну.
